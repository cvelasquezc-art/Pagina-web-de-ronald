# Pagina-web-de-ronald
Pagina web tipo lookbook de gta vi
Pagina web con header, main, section, article, aside y footer.
Utiliza una display grid para crear una grilla de 6 columnas donde los elementos se traslapen
Implementa una paleta de colores definida mediante variables CSS
Integra fuentes externas, utilizando reglas de emparejamiento tipografico.
